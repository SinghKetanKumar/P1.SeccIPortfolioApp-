# Portfolio Management Application

## Overview

This application is built using ASP.NET Core and Blazor Web App.

Features:
- Portfolio List
- Portfolio Details
- REST API
- Dependency Injection
- Swagger Integration
- Postman Testing

## Technology Stack

- .NET 8
- ASP.NET Core Web API
- Blazor
- C#
- Dependency Injection
- Swagger

## API Endpoints

### Get All Portfolios

GET /api/portfolio

### Get Portfolio By Id

GET /api/portfolio/{id}

Example:

GET /api/portfolio/1001

## Running the Application

1. Clone repository
2. Open solution in Visual Studio
3. Build solution
4. Run application
5. Open Swagger

https://localhost:7051/swagger
